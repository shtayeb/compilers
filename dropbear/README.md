# Dropbear Programming Language
A very simple programming language with a REPL and compiler to JavaScript.

## Syntax Example
```
(define x 2)

(add x 3)
```

## Getting started

- Install node dep
```shell
npm install
```
- Run the REPL  
```shell
npm start
```
